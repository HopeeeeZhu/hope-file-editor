# Hope 编辑器

一款轻量、免费、无需登录的 macOS 本地文档阅读与编辑工具。短文可直接阅读或编辑，长文可切换为左右双页的书本模式。

[下载最新版 v0.6.3](https://github.com/HopeeeeZhu/hope-file-editor/releases/latest)

> 当前安装包适用于 Apple 芯片 Mac（M1、M2、M3、M4 等），需要 macOS 10.13 或更高版本。暂不提供 Intel Mac 版本。

![Hope 编辑器界面](docs/images/hope-editor-overview.jpg)

## 一分钟安装

1. 打开上方“下载最新版”页面。
2. 在页面底部的 **Assets** 中下载 `hope-editor-v0.6.3-larger-book-apple-silicon.dmg`。
3. 双击下载好的 `.dmg` 文件。
4. 把“hope的编辑器”拖入“应用程序（Applications）”文件夹。
5. 安装完成后，可以推出安装盘并删除 `.dmg` 文件。

### 第一次打开

应用目前没有 Apple 开发者签名，因此 macOS 第一次打开时可能提示无法验证开发者。这不代表安装失败。

1. 打开“访达 → 应用程序”。
2. 右键点击“hope的编辑器”，选择“打开”。
3. 在弹窗中再次点击“打开”。

如果仍被拦截，进入“系统设置 → 隐私与安全性”，找到刚刚被阻止的应用，点击“仍要打开”。

极少数情况下仍无法启动，可以打开“终端”，粘贴并运行：

```bash
xattr -cr '/Applications/hope的编辑器.app'
```

然后回到“应用程序”再次双击打开。

## 三分钟上手

### 打开已有文档

- 点击顶部“打开”，选择一个 Markdown 或文本文件；也可以按 `⌘O`。
- 安装后，可直接在访达里双击 `.md` 文件打开。
- 打开一个文件后，左侧会显示它所在文件夹的文档目录，点击文件名即可切换。

### 新建、编辑和保存

1. 点击顶部“新建”。
2. 点击“编辑”，显示 Markdown 源文和实时预览。
3. 输入内容；中间预览会同步更新。
4. 点击“保存”或按 `⌘S`，选择文件位置和名称。

文件有未保存修改时，标签名前会出现 `●`。关闭文档时，应用也会询问是否保存。

### 切换阅读与编辑

- 点击“编辑”：显示源文编辑区。
- 点击“关闭编辑”：隐藏源文，进入更清爽的阅读模式。
- 编辑模式下可拖动分隔线，调整文件目录、源文和预览区域的宽度。

### 长文双页阅读

- 点击右上角“长文”，文档会自动排入摊开的左右两页。
- 切换长文时，左侧文件目录和右侧文章大纲会自动收起，尽量展示更多内容。
- Mac 触控板双指左滑向后翻页，双指右滑向前翻页。
- 也可使用页面边缘、左右方向键、Page Up / Page Down 和空格键翻页。
- 宽屏下双页会尽量铺满阅读区域；窗口变窄时会自动切换为单页。

### 搜索内容

- 搜索当前文档：按 `⌘F`，或使用右上角搜索框。
- 按 `Enter` 跳到下一个结果，按 `Shift + Enter` 回到上一个结果。
- 打开文档后，点击顶部“全文搜索”，可搜索当前文件夹内的 `.md` 和 `.txt` 文件。

### 同时处理多份文档

- 每个打开的文档会显示为一个标签页。
- 点击标签页切换文档；拖动标签页可以调整顺序。
- 点击标签页上的 `×`，或按 `⌘W` 关闭当前文档。

### 调整显示

- “缩小 / 放大”：调整编辑区、预览区和目录的字号，应用会记住设置。
- “夜间 / 日间”：切换明暗主题，应用会记住设置。
- 右侧文章大纲默认隐藏，需要时点击顶部“大纲”展开，再点击标题即可跳到对应章节。

## 常用快捷键

| 快捷键 | 作用 |
| --- | --- |
| `⌘O` | 打开文件 |
| `⌘S` | 保存当前文件 |
| `⌘F` | 搜索当前文档 |
| `Enter` | 下一个搜索结果 |
| `Shift + Enter` | 上一个搜索结果 |
| `⌘W` | 关闭当前文档 |

## 支持的内容

可直接打开 Markdown、TXT、PDF 和 HTML 文档。Markdown 与 TXT 可编辑、保存；PDF 使用原页面双页阅读；HTML 经过安全过滤后只读展示。

预览支持常用 Markdown 内容，包括标题、列表、表格、引用、链接、图片、任务清单、脚注、代码高亮、KaTeX 数学公式和 Mermaid 图表。

## 常见问题

### 怎么确认自己的 Mac 能不能安装？

点击屏幕左上角“ → 关于本机”。如果“芯片”一栏显示 Apple M1、M2、M3、M4 或后续 Apple 芯片，即可使用当前安装包；显示 Intel 的 Mac 暂不支持。

### 双击 `.md` 文件时没有使用 Hope 编辑器

在访达中右键一个 `.md` 文件，选择“显示简介”，在“打开方式”中选择“hope的编辑器”，再点击“全部更改”。

### 外部程序也修改了正在编辑的文件

如果当前文档没有未保存内容，Hope 编辑器会自动刷新；如果你同时做了修改，界面会提示选择“重新载入”或“保留当前编辑”。

### 是否必须联网？文件会上传吗？

不需要登录，也不要求联网。文档的读取、编辑和保存都在本机完成；点击文档中的网页链接时，会交给系统默认浏览器打开。
